This is a basic program to make a functional calculator running on user's input uaing the " switch " statement.



Recomended IDE : Turbo C++

*Note: \n and endl ends the line and starts a new one in the output screen, \t gives a tab space before text.

              
    #include <iostream.h>
    #include <conio.h>
    #include <stdio.h>

    void main()
    {
	    clrscr();

	    int choice;
	    float num1,num2,result;

	    cout << "This is a calculator :)";

	    cout << "\n1.Addition.\n2.Subtraction.\n3.Multiplication.\n4.Division.";
	    cout << "\nEnter your choice:";
	    cin >> choice;

	    cout << "Enter first number:";
	    cin >> num1;

	    cout << "Enter second number:";
	    cin >> num2;

	    switch(choice)
	    {
		    case 1:
			result = num1 + num2;
			cout << "The sum is:"<< num1 << "+" << num2 << "=" << result;
		    break;

		    case 2:
			    result = num1 - num2;
			    cout << "The subtracted value is:" << num1 << "-" << num2 << "=" << result;
		    break;

		    case 3:
			    result = num1 * num2;
			    cout << "The multlipicated value is:" << num1 << "*" << num2 << "=" << result;
		    break;

		    case 4:
                            if( num2 !=0)
                            {
			            result = num1/num2;
			            cout << "The division value is:" << num1 << "/" << num2 << "=" << result;
                            }
                            else
                            {
                                cout << "Error: Division by zero is not allowed.";
                            }
		    break;

		    default:
			    cout << "Invalid choice";
		    break;
	    }
	getch();
}


How the program works?

  1. The program displays a menu with options for basic arithmatic operations.
  2. The user enters their choice (1-4) and the two numbers.
  3. The " switch " statement checks the value of the " choice " entered by the user and performs the corresponding operation.
  4. The program handles division by zero and invalid choices.

Example Outputs:

  • If the user chooses addition and enters 10 and 20, the program will display "The sum is : 10 + 20 = 30".
  • If the user chooses division and enters 10 and 0, the program will display "Error: Division by zero is not allowed".
  • If the user chooses an option which is not listed, the program will display "Invalid choice".

Summary:

This program demonstrates the use of a " switch " statement to handle multiple cases and perform different operations based on user input.
