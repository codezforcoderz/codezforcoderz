Program to Calculate the area of a circle:



Recomended IDE : Turbo C++

              
    #include <iostream.h>
    #include <conio.h>
    #include <stdio.h>

    void main()
    {
	    clrscr();

      float r,area;

      cout << "Enter the area of the circle:";
      cin >> r;

      area = 3.14 * r * r;

      cout << "The area of the circle is:" << area;

      getch();
    }


How the program works?

  1. After including the header files we have declared two variables r and area using float data-type.
  2. We asked the user to input the radius of the circle and stored the value inside the variable r.
  3. area = 3.14 * r * r; calculates the area of the circle and stores the value in variable area.
  4. cout >> area; displays the result to the user.

Example Output:

  • If the user enters r = 5, the program will display the result: 78.5.



Program to calculate the area of a triangle:



              
    #include <iostream.h>
    #include <conio.h>
    #include <stdio.h>

    void main()
    {
	    clrscr();
      
      float b,h,area;

      cout << "Enter the length of the base of the triangle:"
      cin >> b;

      cout << "Enter the height of the triangle:";
      cin >> h;

      area = (b * h) / 2;

      cout << "The area of the triangle is:" << area;

      getch();
    }


How the program works?

  1. After including the header files we have declared three variables b, h and area using float data-type.
  2. We asked the user to input the base and height of the triangle and stored the value inside the variables b and h respectively.
  3. area = (b * h) / 2; calculates the area of the triangle and stores the value in variable area.
  4. cout >> area; displays the result to the user.

Example Output:

  • If the user enters 5 as base and 6 as height, the program will display the result: 15.
