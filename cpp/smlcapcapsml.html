Basic program to Convert Capital letters to Small letters:



Recomended IDE : Turbo C++

              
        #include <iostream.h>
        #include <stdio.h>
        #include <conio.h>
        void main()
        {
	        clrscr();
	        char alpha;
	        cout << "Enter any capital letter :" ;
	        cin >> alpha;
	        if(alpha >= 65 && alpha <= 97)
	            {
	                alpha = alpha + 32 ;
	            }
	        cout << "After the conversion the letter will be :" << alpha ;
	        getch();
    	}


How the program works?

  1. We declare a character variable alpha to store the input letter.
  2. We prompt the user to enter a Capital letter.
  3. We use the ASCII value difference between small and capital letters (32) to convert the Capital letter to Small letter.
  4. We display the Small letter.

ASCII value difference:

  • Small letters 'a' to 'z' have ASCII values 97 to 122.
  • Capital letters 'A' to 'Z' have ASCII values 65 to 90.
  • The difference between the ASCII vslue of small and capital letters is 32 (97-65 = 32).

Example output:

  • If the user enters 'A', the program will take the ASCII value of 'A' which is 65 and add 32 to it to get 97 which is the ASCII value of 'a'.
  • If the user enters 'a'(97), the program will not run and show error because inside the " if " statement we have declared the input range from 65 to 97.

Basic program to convert Small letters to Capital letters.



              
        #include <iostream.h>
        #include <stdio.h>
        #include <conio.h>
        void main()
        {
	        clrscr();
	        char alpha;
	        cout << "Enter any small letter :" ;
	        cin >> alpha;
	        if(alpha >= 97 && alpha <= 122)
	            {
	                alpha = alpha - 32 ;
	            }
	        cout << "After the conversion the letter will be :" << alpha ;
	        getch();
    	}


How the program works?

  1. We declare a character variable " alpha " to store the input letter.
  2. We prompt the user to enter a Small letter.
  3. We check if the input character is a small letter.
  4. If it is a small letter, we subtract 32 from it's ASCII value to convert it to the corresponding capital letter.
  5. We display the capital letter.

Example output:

  • If the user enters 'a', the program will take the ASCII value of 'a' which is 97 and subtract 32 from it to get 65 which is the ASCII value of 'A'.
  • If the user enters 'A'(65), the program will not run and show error because inside the " if " statement we have declared the input range from 97 to 122.

Summary:

The programs above will work for any small or capital letter and will correctly convert into the corresponding capital or small letter.
If you enter a non-alphabet character, it will be displayed as it is.
