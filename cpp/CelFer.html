This is a basic program to convert celsius to farenheit.



Recomended IDE : Turbo C++

              
    #include <iostream.h>
    #include <conio.h>
    #include <stdio.h>

    void main()
    {
	    clrscr();

      int c,f;

      cout << "Enter the temparature in celsius:"
      cin >> c;

      f = (c * 1.8) + 32;
      
      cout << "The temparature in farenheit is:"<< f << "farenheit.";
      
      getch();
    }


How the program works?

  1. After including the header files we declared two variables c and f using int data type.
  2. We asked user to input the temparature value in celsius then took the input using cin >> c; and stored the value in variable c.
  3. f = (c * 1.8) + 32; calculates the conversion of celsius to farenheit and stores the value in the variable f.
  4. cout << f; displays the answer to the user.

Example Output:

  • If the user enters 0 celsius the program will calculate the output and show 32 farenheit.
  • If the user enters 35 celsius the program will calculate the output and show 92 farenheit.
