This is a basic program to calculate the grades of a student using the " switch " statement.



Recomended IDE : Turbo C++

              
                #include <iostream.h>
                #include <stdio.h>
                #include <conio.h>

                void main()
                {
                        clrscr();

                        char grade;

                        cout << "Enter grade (A-F):";
                        cin >> grade;

                        switch(grade)
                        {
                            case 'A':
                            case 'a':
                                cout << "Excellent! Your score is 90-100.";
                            break;

                            case 'B':
                            case 'b':
                                cout << "Good job! Your score is 80-89."
                            break;

                            case 'C':
                            case 'c':
                                cout << "Fair. Your score is 70-79."
                            break;

                            case 'D':
                            case 'd':
                                cout << "Poor. Your score is 60-69."
                            break;

                            case 'F':
                            case 'f':
                                cout << "Shouldn't have taken that test."
                            break;
                            
                            default:
                                cout << "Invalid grade."
                            break;
                        }
                    getch();
                }


How the program works?

  1. The program asks the user to enter a grade.(A-F)
  2. The " switch " statement checks the value of the " grade " variable.
  3. Based on the value of " grade ", the program prints a message indicating the corresponding score range.
  4. The program handles both uppercase and lowercase grades.
  5. If the user enters an invalid grade, the program prints "Invalid grade".

Example Outputs:

  • If the user enters " A " or " a ", the program will print "Excellent! Your score is 90-100.
  • If the user enters " E " or " e ", the program will print " Invalid grade ".

Summary:

This program dedmonstrates the use of a " switch " statement with multiple cases and default handling in C++.
