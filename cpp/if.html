This is a simple program in Turbo C++ that uses a " if " statement without an " else " clause to print a message if the user entered number is positive.



Recomended IDE : Turbo C++

              
                #include <iostream.h>
                #include <stdio.h>
                #include <conio.h>

                void main()
                {
                        clrscr();

                        int num;

                        cout << "Enter a number: ";
                        cin >> num;

                        if (num > 0)
                        {
                            cout << "The number is positive";
                        }
                        getch();
                }


How the program works?


  1. The program asks the user to enter a number. ( cout << "Enter a number:";)
  2. The " if " statement checks whether the number is greater than 0. (if (num > 0) )
  3. If the condition is true, the program prints "The number is positive." ( cout << "The number is positive")
  4. If the condition is false, the program does nothing and waits for a key press.

Example Output:


  • If the user enters, a number greater than 0, suppose 5, the program will print "The number is positive".
  • If the user enters 0, or a negative number, suppose -5, the program won't do anything and it'll wait for a key press to go back.

Summary:

This program demonstrates the basic use of an " if " statement without an " else " clause in Turbo C++.
If the condition is not met, the program simply continues executing without taking any action.
