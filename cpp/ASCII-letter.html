Basic program to Convert ASCII code to Letter:



Recomended IDE : Turbo C++

              
        #include <iostream.h>
        #include <conio.h>
        #include <stdio.h>

        void main()
        {
	        clrscr();

	        int value;

	        cout << "Enter the ASCII value:" ;
	        cin >> value;

	        if( value >= 65 && value <=122 )
	        {   
		        cout << "The corresponding letter is:" << (char)value;
	        }
	        else
	        {   
		        cout << "This ASCII value is not convertable";
	        }
    	    getch();
        }



How the program works?

  1. We declare an integer variable "value" to store the input ASCII code.
  2. We prompt the user to enter an ASCII code.
  3. We check if the input ASCII code corresponds to a printable character.(32 to 126)
  4. if it's a printable character, we convert the ASCII code to a character using a simple assignment.
  5. We display the character corresponding to the ASCII code.

Example OutPut:

  • If the user enters '66', the program will display 'B'.
  • If the user enters '126', the program will display '~'.

Casting to 'char':

The '(char)' cast tells the compiler to treat the char value as an 'int', effectively giving us the ASCII code of the character.

Summary:

This program will work for any ASCII code input and will correctly display the corresponding character if it's printable.
If the ASCII code does not correspond to a printable character, it will display a message indicating that.



Program to convert letters to ASCII code:



              
        #include <iostream.h>
        #include <conio.h>
        #include <stdio.h>

        void main()
        {
            clrscr();

            char ch;

            cout << "Enter any Small or Capital letter:";
            cin >> ch;

            cout << "The ASCII value is:" << (int)ch;
            getch();
        }


How the program works?

  1. We have declared a character variable 'ch' to store the input character.
  2. We prompt the user to enter a character.
  3. We convert the character to it's ASCII code using a simple cast '(int)ch'.
  4. We display the ASCII code of the character.

Example OutPut:

  • If the user enters 'C', the program will display '67'.
  • If the user enters 'z', the program will display '122'.

Casting to 'int':

The '(int)' cast tells the compiler to treat the 'char' value as an 'int', effectively giving us the ASCII code of the character.


Summary:

This program will work for any single character input and will correctly display it's ASCII code.
