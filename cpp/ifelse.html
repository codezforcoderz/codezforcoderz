This is a simple program in Turbo C++ that uses an " if-else " statement to determine whether a student has passes or failed based on their obtained marks.



Recomended IDE : Turbo C++

              
                #include <iostream.h>
                #include <stdio.h>
                #include <conio.h>
                
                void main()
                {
                    clrscr();

                    int marks;

                    cout << "Enter your marks:";
                    cin >> marks;

                    if (marks >= 40)
                    {
                        cout << "You have passed.";
                    }
                    else
                    {
                        cout << "You have failed.";
                    }
                    getch();
                }


How the program works?


  1. The program asks the user to enter their obtained marks. (cout << "Enter your marks:";)
  2. The " if " statement checks whether the marks are greater than or equal to 40. (if (marks >= 40) )
  3. If the condition is true, the program prints "You have passed".
  4. If the condition is false, the program will execute the " else " part and "You have failed" will be displayed.

Example Output:


  • If the user enters 50, the program will print "You have passed".
  • If the user enters 30, the program will print "You have failed".
Summary:

This program demonstrates the basic use of an " if-else " statement in C++. This statement allows the program to make a decision based
on a condition and execute different blocls of code accordingly.
