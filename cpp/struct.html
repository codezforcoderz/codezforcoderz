First we will take a program as an example:


This is a simple program to print "Hello world in C++. We will take this program as an example to understand the structure of the program.

              
                #include<iostream.h>
                #include<conio.h>
                #include<stdio.h>
                void main()
                {
                  clrscr();

                  cout << "Hello world";

                  getch();
                }

So, the structure of the program starts with:

  1. Processing directives
    • Header files:
      • #include<iostream.h>
      • #include<conio.h>
      • #include<stdio.h>

      These are called the "Header files" of a program which are the most important things to execute the program properly.


  2. Main function
    • Entery point of the program.

      • Here void main() is the "Main function" of the program and the program codes are written inside this main function using { }(curly brackets).


  3. Statements and expressions
    • Perform calculations, input/output, operations and control flow.

      • Here, cout<< "Hello world" is used to display the text written inside the double quotations.

At last we've used "getch();" to end the program and what is does is pause the program execution and keep the console window open until the user presses any key.


So, the basic structure of a C++ program looks like this:

  • #include (header files)
    main function()
    {
    program code;
    statements and executions;

    getch();
    }

This structure provides a foundation for building C++ programs. As you progress, you'll learn more about each component and how to use them effectively.

Note: Use clrscr(); to clear the output screen before running the program.
